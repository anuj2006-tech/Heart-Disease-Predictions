 Heart Disease Prediction System

A Machine Learning-based web application that predicts the possibility of heart disease using patient health parameters. The application is built with **Streamlit** and uses a trained **K-Nearest Neighbors (KNN)** model for real-time prediction.

 Features

- User-friendly Streamlit interface
- Real-time heart disease prediction
- Data preprocessing and feature scaling
- Integrated Machine Learning model
- Fast and interactive results

 Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Joblib
